# familyhelper-ui

## 项目简介

家庭助手系统的前端。

该项目提供一个标准的网页前端，供用户进行交互。

---

## 特性

- 基于 maven 进行项目托管，可一键打 war 包，使用方便。
- view 模块基于 pnpm + monorepo，实现项目内功能模块最大化复用。
- 使用 vue3 + 自研 vim（Vue Interactive Model）构建 web 应用，最大化优化开发体验。
- 基于自研打包脚本，在 monorepo 项目打包时，多包并行打包，并能正确处理依赖关系。
